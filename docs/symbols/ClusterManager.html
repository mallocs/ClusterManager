Class Detail
ClusterManager(map, opts)
Creates a new Cluster Manager for clustering markers on a V3 Google map.
Parameters:
{GMap3} map
The map that the markers should be added to.
{object} opts Optional
Options for configuring the behavior of the clustering. Defaults are applied in resetManager.
{google.maps.Marker[]} opts.markers Optional
Markers to add to the manager.
{function} opts.zoom_to_precision Optional, Default: function(zoom_level) {return zoom_level + precision;}
A function to set the precision for each zoom level.
{number} opts.precision Optional, Default: 2
A number between 0 and 27 that sets how small the cluster boxes will be. Higher numbers will make smaller boxes.
{string|object} opts.icon_color Optional, Default: "00CC00"
Sets the default icon color in HEX. Default is a bright green.
{number} opts.padding Optional, Default: 200
The amount of padding in pixels where markers not in the viewport will still be added to the map.
{boolean} opts.visualize Optional, Default: false
For debugging. Will put a box around each cluster with at least one marker.
{number} opts.cluster_by_distance Optional, Default: true
Combine neighboring clusters if they are close together. This is a little slower but makes more rounded clusters.
{number} opts.cluster_distance_factor Optional, Default: 2048000
Clusters are combined if they are within this distance: cluster_distance_factor*Math.pow(2, -precision+2)

{string} getGeohash(lat, lng, precision)
Gets a hash based on latitude, longitude and precision. Higher precisions are geographically smaller areas. Since distance between degrees of longitude varies based on latitude: (pi/180)*(6,378,137.0 meters)cos(degrees latitude) the area covered by a given geohash precision will get smaller as it approaches the poles (cos(90 degrees) = 0). If you visualize the boxes, however, they will look larger based on the map projection.
